# Conversations Log

This folder records every tutor session — what was discussed, what was decided, what was taught, and what feedback was given. Nothing gets lost.

## Why This Exists

- Conversations between tutor and learner contain decisions, clarifications, and context that lesson files alone don't capture
- The tutor can refer back to prior sessions to maintain continuity and catch contradictions
- Anyone following this learning path can read through the conversations to understand the "why" behind decisions
- Acts as a living FAQ — questions asked once don't need to be re-answered

## File Naming

Each session is logged as `YYYY-MM-DD-topic.md` (e.g., `2026-04-25-project-setup.md`).

If multiple sessions happen on the same day, a suffix is added: `2026-04-25-lesson-01-beginner.md`, `2026-04-25-lesson-01-intermediate.md`.

## What Gets Logged

- Key decisions made and their rationale
- Concepts taught and examples discussed
- Tasks attempted, feedback given, and corrections made
- Contradictions or changes in approach (flagged explicitly)
- Best practices introduced outside of lessons
- Questions asked by the learner and answers given
